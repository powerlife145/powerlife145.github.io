3) JavaScript & JQuery 반복문

let alphabet = ['a', 'b', 'c']
console.log(alphabet[0])
console.log(alphabet[1])
console.log(alphabet[2])
이래야 하는 것을
```java
let alphabet = ['a', 'b', 'c']
alphabet.forEach((a) => {
console.log(a)
})
```
